# DPFTS Prompt Examples

Use these prompts after installing DPFTS. Replace names, currencies, item IDs, and systems with your own game details.

## Security

- "Use DPFTS to review this RemoteEvent system like an exploiter would."
- "Use DPFTS to audit my trading system for dupes, client trust, and leave/disconnect bugs."
- "Use DPFTS to find every place where the client controls currency, inventory, damage, rewards, or admin actions."
- "Use DPFTS to design server-side validation for this teleport request."

## Code And Architecture

- "Use DPFTS to structure my Roblox project so server, client, remotes, configs, and services are clean."
- "Use DPFTS to refactor this giant script into practical ModuleScripts without overengineering."
- "Use DPFTS to build a full Roblox system. Include file paths, Studio placement, server/client separation, code, tests, and failure cases."
- "Use DPFTS to review whether this Roblox system is too complicated for an MVP."

## DataStores

- "Use DPFTS to design a safe DataStore profile system for Coins, Level, Inventory, and Settings."
- "Use DPFTS to review this DataStore code for missing defaults, schema validation, save spam, and failure cases."
- "Use DPFTS to add save slots without letting players corrupt or overwrite each other's data."

## UI And Design

- "Use DPFTS to improve this Roblox UI without adding button spam."
- "Use DPFTS to design mobile-friendly Roblox buttons for my shop UI."
- "Use DPFTS to review my Roblox place layout and tell me what feels like AI slop."
- "Use DPFTS to design a first-time tutorial that reaches the first reward quickly."
- "Use DPFTS to review my thumbnail idea and check whether it honestly represents the game."

## Economy And Monetization

- "Use DPFTS to review my Roblox economy for inflation, bad sinks, and pay-to-win risk."
- "Use DPFTS to add a safe gamepass perk without trusting the client."
- "Use DPFTS to build a daily reward that cannot be spammed."
- "Use DPFTS to review this developer product receipt flow for duplicate grants."

- "Use DPFTS to build a simple server-owned inventory with equip requests."
